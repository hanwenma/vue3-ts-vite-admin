# 启动项目
- 使用 `npm install` 依赖安装
- 使用 `npm run dev` 快速启动项目 

# eslint 一键修复
- 使用 `npm run lint` 启用 `eslint` 自动修复

# prettier 一键格式化
- 使用 `npm run prettier` 启用 `prettier` 自动格式化
# 代码提交
- 使用 `git add .` 将修改的文件添加到暂存区
- 使用 `npm run commit` 将暂存区的文件进行提交
- 使用 `git push` 将提交内容推送到远程
